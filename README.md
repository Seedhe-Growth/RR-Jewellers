# Saira Ornaments / The Jewel Vault

## Project Synopsis
**Saira Ornaments** (The Jewel Vault) is a premium, full-stack MERN (MongoDB, Express, React, Node.js) e-commerce and portfolio platform designed for a luxury jewellery boutique. The project focuses on bridging the gap between high-end editorial storytelling and seamless digital commerce, catering to a modern, Instagram-first audience.

## Key Features
- **Premium UI/UX**: Minimalist Black & Gold aesthetic with glassmorphism and Framer Motion animations.
- **Dynamic Catalog**: Full product listing with advanced filtering (Material, Occasion, Category) and search.
- **State-of-the-art Shop**: Persistent cart (Zustand), wishlist, and high-fidelity product zoom.
- **Admin Command Center**: Complete dashboard for managing inventory, category CRUD, order tracking, and customer inquiries.
- **High Trust Signals**: Integration for WhatsApp concierge, BIS Hallmarking badges, and brand storytelling pages.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Zustand.
- **Backend**: Node.js, Express, JWT Authentication, Multer + Cloudinary.
- **Database**: MongoDB Atlas with Mongoose ODM.

## Getting Started
1. **Server**: `cd server && npm install && npm run dev`
2. **Client**: `cd client && npm install && npm run dev`
